Q
역전파(Backpropagation) 알고리즘에서 경사 소실(Vanishing Gradient) 문제가 발생하는 주요 원인은?
딥러닝 기초 틀림 4회 최근: 2026.03.31
정답: ③ Sigmoid 활성화 함수 사용 시 기울기가 0에 수렴

Sigmoid 함수는 양쪽 끝단에서 포화(saturation)되어 기울기가 거의 0에 가까워집니다. 이 값이 역전파 과정에서 반복 곱해지면 층이 깊어질수록 기울기가 지수적으로 감소합니다. 이를 해결하기 위해 ReLU 활성화 함수와 배치 정규화(Batch Normalization)가 주로 사용됩니다.
Q
F1-Score를 사용하는 것이 가장 적합한 상황은?
모델 평가 틀림 3회 최근: 2026.04.01
정답: ② 클래스 불균형이 심하고 정밀도와 재현율이 모두 중요한 경우

F1-Score는 Precision과 Recall의 조화 평균입니다. 클래스 비율이 불균형하면 Accuracy는 다수 클래스에 편향되므로 F1-Score가 더 유효한 지표입니다.
Q
드롭아웃(Dropout) 기법의 적용 방식과 추론 단계에서의 처리 방법으로 옳은 것은?
딥러닝 기초 틀림 2회 최근: 2026.03.29
정답: ④ 훈련 시 무작위로 뉴런 비활성화, 추론 시 모든 뉴런 활성화 후 드롭아웃 비율만큼 스케일 조정

드롭아웃은 훈련 중 랜덤하게 뉴런을 비활성화하여 과적합을 방지합니다. 추론 단계에서는 모든 뉴런을 활성화하되 출력에 (1-p)를 곱하거나 inverted dropout을 사용합니다.
Q
k-평균 클러스터링(K-Means)에서 k 값 결정에 사용하는 엘보우(Elbow) 방법의 기준 지표는?
머신러닝 기초 틀림 2회 최근: 2026.03.28
정답: ① WCSS(Within-Cluster Sum of Squares, 군집 내 분산 합)

엘보우 방법은 k를 증가시키면서 WCSS의 감소 속도가 급격히 줄어드는 지점(꺾이는 부분)을 최적 k로 선택합니다.
