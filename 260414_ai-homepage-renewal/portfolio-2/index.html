Core Capabilities

핵심 기술 역량

최신 웹 기술을 활용하여 인상적인 인터랙션과 뛰어난 성능을 동시에 구현합니다.

🎯
패럴랙스 인터랙션
mousemove + rAF 기반 레이어별 깊이감 있는 패럴랙스 효과를 구현하여 생동감 있는 히어로 섹션을 제공합니다.
🖼️
갤러리 라이트박스
React Portal 기반 body 마운트 라이트박스로 z-index 충돌 없이 확대 미리보기와 키보드 내비게이션을 지원합니다.
👆
드래그 슬라이더
Pointer Events API 단일 인터페이스로 마우스·터치·스타일러스를 통합 처리하는 범용 슬라이더를 설계했습니다.
📊
스크롤 진행 인디케이터
Intersection Observer 10단계 threshold로 섹션 진입률을 감지하여 실시간 스크롤 진행 바를 구현합니다.
⚡
60fps 렌더링
requestAnimationFrame 루프와 will-change 최적화로 애니메이션 성능을 유지하는 구조를 설계했습니다.
♿
접근성 지원
키보드 포커스 관리, ARIA 속성, WCAG 대비비 기준을 준수하여 모든 사용자를 위한 인터페이스를 구현합니다.
