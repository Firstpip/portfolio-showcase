자주 묻는 질문
게임을 처음 시작하려면 어떻게 해야 하나요?
공식 홈페이지 다운로드 페이지에서 게임 클라이언트를 무료로 설치할 수 있습니다. 설치 후 회원가입을 통해 계정을 생성하면 즉시 플레이 가능합니다. 최소 시스템 사양은 다운로드 페이지에서 확인하세요.
랭킹 시스템은 어떻게 운영되나요?
랭킹은 시즌 기준으로 초기화됩니다. 각 시즌의 점수는 승/패/개인 기여도를 종합하여 산출되며, 시즌 종료 시 최종 랭크에 따른 보상이 지급됩니다. 브론즈 → 실버 → 골드 → 플래티넘 → 다이아몬드 → 마스터 순입니다.
핵(치트) 신고는 어떻게 하나요?
게임 내 신고 기능을 통해 해당 플레이어를 실시간으로 신고할 수 있습니다. 신고 내용은 보안팀에서 24시간 이내 검토하며, 사실 확인 시 즉시 제재 조치가 이루어집니다. 허위 신고는 신고자 계정에도 불이익이 있을 수 있습니다.
계정 정보를 잊어버렸을 때 어떻게 하나요?
로그인 페이지의 '비밀번호 찾기'를 통해 가입 시 등록한 이메일로 재설정 링크를 받을 수 있습니다. 이메일 접근이 불가한 경우 1:1 문의를 통해 본인 인증 후 계정 복구를 지원합니다.
게임 업데이트는 언제 진행되나요?
정기 업데이트는 매월 첫째 주 수요일 오전 9시~12시에 진행됩니다. 긴급 패치는 사전 공지 없이 진행될 수 있으며, 점검 전 공지사항을 통해 안내됩니다. 게임 클라이언트는 점검 후 자동 업데이트됩니다.
